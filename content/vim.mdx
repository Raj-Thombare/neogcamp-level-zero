# Vim - Vi improved

link - https://www.youtube.com/watch?v=pTIfeeqd6VI

### Three modes in vim -

1. Normal Mode
2. Visual Mode
3. Insert Mode

### Exit Vim -

:q - exit the vim

### Basic Navigation -

H - Left

J - Bottom

K - Top

L - Right

### Copy, Delete and Paste

Delete - dd

Copy - yy

Paste - p

To go 100 line in code - :100

Skipping blocks ( { } )

### Undo and Redo

Undo - u

Redo - ctrl + r

### Navigate to start and end in line

Go at the end of line - shift + $

Go at the start of the line - 0

Period for the most recent text - .
